***HOW  TO  RUN  ON  MENTOR  GRAPHICS  -  QUESTA  SIM***

1.  Create a new project
2.  Add all the files with **.sv** extension
3.  Compile all the files
4.  It's perfectly fine and not all the files will be successfully compiled
5.  Now compile only the top testbench file i.e. **testbench.sv**
6.  Now start the simulation
7.  From the work folder , select the **testbench** module and run
8.  Select the **intff** from the left panel
9.  All the ports a,b,c,sum and carry will be there
10.  Select them all and add them to the wave by right clicking them
11.  Press the run button
